Git Cheatsheet

git config --global user.name blabla

Sets name of the user, who changes files in repository so that others knew, who made changes

git config --global user.email blabal@bla

Sets email of the user, who changes files in repository so that others knew, who made changes

git init

Initiates local git/ creates local repository

git status

Lists all new/changed files

git add

Adds/stages files for commitment

git commit

Saves added files/changes to the repository

git log

Displays commit history/logs

git checkout

Allows to move to the desired log or branch

git branch

Allows to create and delete branches

git push

Updates remote repository with the changes commited in local repository

git pull

Downloads from remote repository or different branch

git merge

Combines changes and updates from different repositories/branches

git clone

Copies a branch or a repository

.gitignore file

Contains a list of files/extensions etc that should not be considered by git (those are not added/commited or merged)

git revert

"Undo" some of the recent commits

git reset

Resets current working branch

git rebase

Compresses all the changes into a single tree line

git cherry-pick

Applies changes to some ("picked") commits

git stash

Allows to temporarily store the changes, but without commiting them yet
